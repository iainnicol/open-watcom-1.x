             Overview of all Watcom C/C++/Fortran ZIP archives			 b
             =================================================

Files common to both Watcom C/C++ and Watcom Fortran 77
=======================================================
cm_core_all	  - Core binaries (All hosts)
cm_core_dos	  - Core binaries (DOS host)
cm_core_win	  - Core binaries (Win16 host)
cm_core_os2	  - Core binaries (OS2 host)
cm_core_nt	  - Core binaries (NT host)
cm_core_doswin	  - Core binaries (DOS & Win hosts)

cm_clib_hdr       - C runtime library header files

cm_clib_a16       - C runtime libraries (16-bit, all targets)
cm_clib_a32       - C runtime libraries (32-bit, all targets)
cm_clib_d16       - C runtime libraries (16-bit DOS target)
cm_clib_d32       - C runtime libraries (32-bit DOS target)
cm_clib_n32       - C runtime libraries (32-bit Netware target)
cm_clib_o16       - C runtime libraries (16-bit OS/2 target)
cm_clib_o32       - C runtime libraries (32-bit OS/2 target)
cm_clib_w16       - C runtime libraries (16-bit Windows & Win386 targets)
cm_clib_w32       - C runtime libraries (32-bit Windows target)

cm_plib_a16       - C++ runtime libraries (16-bit, all targets)
cm_plib_a32       - C++ runtime libraries (32-bit, all targets)
cm_plib_o32       - C++ runtime libraries (32-bit, OS/2 target)
cm_plib_w32       - C++ runtime libraries (32-bit, Windows target)

cm_dbg_all        - Debugger                     (All hosts)
cm_dbg_nt         - Debugger, profiler & sampler (NT host)
cm_dbg_os2        - Debugger, profiler & sampler (OS/2 host)
cm_dbg_dos        - Debugger, profiler & sampler (DOS host)
cm_dbg_win        - Debugger, profiler & sampler (Win16 host)
cm_dbg_doswin     - Debugger                     (DOS & Win16 hosts)
cm_dbg_dosos2     - Debugger                     (DOS & OS/2 hosts)
cm_dbg_winos2     - Debugger                     (Win16 & OS/2 hosts)
cm_dbg_os2nt      - Debugger                     (OS/2 & NT hosts)
cm_dbg_misc1	  - Debugger			 (DOS host or target)
cm_dbg_misc2	  - Debugger			 (Win16, 95, 98 or ME hosts)
cm_dbg_net	  - Debugger servers & sampler   (Netware host)

cm_hlp_dos	  - Help files (Dos host)
cm_hlp_win	  - Help files (Win16 host)
cm_hlp_os2	  - Help files (OS/2 host)
cm_hlp_nt	  - Help files (NT host)
